## Agentic Speech Studio

Turn any text into lifelike speech directly in the browser with adjustable voices, rate, pitch, and volume controls.

### Getting Started

```bash
npm install
npm run dev
```

Open `http://localhost:3000` to start using the studio.

### Scripts

- `npm run dev` – start the development server
- `npm run build` – create a production build
- `npm run start` – run the production build locally
- `npm run lint` – run ESLint

### Deploy

```bash
vercel deploy --prod --yes --token $VERCEL_TOKEN --name agentic-e8ab5579
```
